## ⭐ 特性


1. 自动滚动获取所有订阅内容。

2. 提供独立面板供用户管理订阅。
   
3. 支持批量勾选取消订阅。
   
4. 支持搜索关键字。

<br>

## 🗨 碎碎念
为什么要在收藏的时候自动订阅合集！经常看到不是我想看的内容出现在更新列表里...

所以这个脚本就诞生啦，无论是管理已有合集，还是直接批量取消订阅所有的合集都是可以的哦~
